# 統計学教科書 章構成案 v7

- 想定読者: 初学者. 数学の前提は最小限.
- 前提: 本書は確率パートの後続文書. 期待値・確率変数・確率分布などの概念と記法は既習として使用可.
- 記述テンプレ: 各章は 6ステップ(問い→素朴な方法と限界→理論の最小核→手法→実践→結果と次の一手) で統一. ただし1章(overview)と6章Part B(Part Aの数学的補遺)は構成上の例外とする.
- 節見出し: 本文の節見出しは, 設計上の6ステップ呼称に縛られない. 読者にとって直接的な見出しを採用してよい. 章ごとに見出しスタイルを統一する義務はない. 設計上の対応関係は本構成案で管理する.
- ランニング例: 「コンテンツの人気と評価」を主軸とする(人気の形成・レビューの信用). 具体的サービス・変数設計は別途確定予定. ECサイト購買データは不採用.
- ランニング例の併用方針: 各章では手法の説明に適したデータを使ってよい. 章末でランニング例に戻り, 学んだ手法を当てはめる. 単一データセットで全章を貫く構成にはしない.
- 表記: 句読点は「, .」で統一.
- 方針: 本文は「統計の骨格」に限定し, 詳細は各章末の「補足概念」に退避.
- LaTeX: \paragraph は使用禁止. 小見出しは \subsubsection* を使用.
- バージョン運用: 本章構成案はメジャーアップデートのみで管理する. 更新のたびに整数で番号を増やす(v6 → v7 → v8 ...).

## 1. 統計的推論という問題(overview)

**章の役割**  
本章は教科書全体のoverviewである. 6ステップ構成を適用せず, 文体・修辞・構成上の制約も外す. 学びの見取り図と問題提起に集中し, 読者を本書の世界へ招き入れることを優先する.

**物語の役割**  
意思決定は待ってくれないのに, 私たちが手にできるのは世界の一部だけ. 「不完全な標本から母集団をどこまで確からしく語れるか」を**自分事**として立てる章. 以降の章が何を担うのか, 学びの見取り図をここで共有する. あわせて「正しく計算しても結論が間違いになる」例を1〜2個だけ先出しし, バイアスという地雷の存在を読者に認識させる. 解決策はまだ出さない.

**データ分析での位置づけ**  
問いの定式化と評価軸の確定. 対象, 単位, 変数, 観測範囲, 既知のバイアス源を文章で固定し, 迷走を防ぐ起点をつくる.

**バイアス先出しの位置づけ**  
全章共通ルール(§6, バイアスの扱い方針)に従い, 「正しく計算しても結論が間違いになる例」を1〜2個先出しする. 9章への伏線として必須.

**節構成と各節の役割**  

1. **わずかな差をしっかりみる**  
   じゃんけんの出現比(35:32:33)を入口として, わずかな差でもデータが十分なら「偶然ではない」と言い切れることを体験させる. 統計的に意味のある差とはどういうことかの初出.
2. **確率論と統計学**  
   確率論(仕組み→データ)と統計学(データ→仕組み)が考える向きが反対であることを示す. 確率パート読了者に対し, 本書がどう接続するかを明示する.
3. **目標をかたちにする**  
   分析の出発点は「自分が何を知りたいか」を言葉にすること. 問いの設定の仕方そのものには踏み込まないが, 読者が常に自分の問いを意識する姿勢を促す. 行きつ戻りつが普通であることも明記.
4. **理解してほしいこと**  
   本書を通して読者に掴んでほしい三つの核心概念(平均・分散・正規分布)を簡潔に宣言する. 各章で個別の到達目標を設定するのではなく, 本書全体の北極星として置く.
5. **分析の準備**  
   標本と母集団の用語を導入. 「完全なデータは存在しない」という統計学の出発点を明示する. ランニング例の併用方針(各章は手法に適したデータを使い, 章末でランニング例に戻る)もここで宣言する.
6. **コラム: データとは何か**  
   数値データとカテゴリデータの区別, 表のデータが本書で扱う範囲であることなど, 用語の地ならしを行う.

**扱う根本概念**  
母集団と標本, 母数と統計量, 偶然変動(ランダムネス), 系統誤差(バイアス).

**補足概念**  
測定尺度, 調査設計の粒度と枠組み.

## 2. データの要約 — 代表値と散らばり
**物語の役割**  
生データの海から, まず「大きさ」と「ばらつき」をつかむ. 平均が同じでも散らばりが違えば判断は変わる, を自分の手で確かめる.

**データ分析での位置づけ**  
EDAの出発点. 欠損や外れの一次点検をしつつ, 以降で使う要約指標の言語を揃える.

**扱う根本概念**  
平均, 中央値, 分位点, 分散, 標準偏差, 四分位範囲(IQR), 外れ値の考え方.

**補足概念**  
最頻値, 変動係数, 切断平均, MAD, 歪度・尖度.

## 3. 実データの分布と可視化
**物語の役割**  
2章で得た統計量という要約は, 集まりの中心や散らばりを数で表す. しかし数だけでは集まりの「形」までは戻ってこない. 本章は形を見ることで, 統計量で失われた構造に気づき, データを分布として捉え直す視点を獲得する章. ヒストグラム・箱ひげ図・散布図で, 歪み・多峰性・群間の差・関係を目で見て仮説を育てる. 4章以降の確率分布・標本分布の話を素直に飲み込めるようにする素地でもある.

**データ分析での位置づけ**  
EDAの中核. 以降の推定・検定・回帰に入る前提(正規近似の見当, 群分けの必要性)を**仮置き**する.

**章内で扱う限界**  
各図の死角を性格として軽く触れる程度に留める. ヒストグラムのビン幅依存, 箱ひげ図で形(特に二峰)が見えにくいこと, 散布図から因果は読めないこと, など. 軸の切り取りや不適切な可視化の体系的な議論は9章に渡す.

**扱う根本概念**  
ヒストグラム, 箱ひげ図, 散布図, 経験分布の発想.

**補足概念**  
密度推定, QQプロット, スモールマルチプルズ.

## 4. 確率分布と中心極限定理 — 推論の足場をつくる
**物語の役割**  
「取り直したら要約値はばらつく」を掴み, そのばらつきが正規へ寄る事情(CLT)を理解する. SEの式を手に入れ, 「±約2SEで幅をつけて報告できる」という到達点で章を閉じる. ここで**区間推定と検定の土台**が完成する.

**データ分析での位置づけ**  
EDAから推論への橋渡し. 正規近似を使ってよい根拠を概念レベルで確立する.

**章の到達点**  
SE = s/√n で推定値のばらつきを測れる. 推定値 ± 約2SE で幅をつけて報告できる. この「幅」の正しい読み方と運用は次章で扱う.

**扱う根本概念**  
理論分布という言語, 正規分布(定義・68-95-99.7ルール), 中心極限定理, 標準誤差(SE)の定義と部品の分解, SD vs SEの区別.

**「標本分布」初出時の扱い**  
本書では本章が「標本分布」の初出位置である. 要件定義書§4.7に従い, 「手元の標本そのものの分布ではなく, 統計量を繰り返し取ったときの分布」であることを明示的に区別する説明を初出時に添える. 以降の章では標準用語として使用してよい.

**章内の実践**  
レビュー評価の平均に±2SEの幅を添える例を1本のみ. 複数ケースの実践や件数別比較は5章に委ねる.

**コラム方針**  
- 理論分布カタログ: 大幅に圧縮する. 後続章で使わない分布の詳述を避け, 「仕組み→形の対応」の骨格のみ示す.  
- ケトレーのコラム: 残す(正規分布を型に押し込むことへの教訓).  
- 統計量の分布の予告コラム(t分布・χ²・F分布): **削除**. 使う前の予告は読者の負荷. 6章で必要時に導入すれば足りる.

**補足概念**  
大数の法則, ブートストラップという保険, 極端に歪な分布での注意.

## 5. 推定 — 信頼区間の読み方と使い方
**物語の役割**  
前章でSEと「幅をつける」発想を手に入れた. 本章はその幅を**どう読み, どう使うか**に集中する. 95%信頼区間の正しい意味を文で説明でき, 平均・割合・差の3ケースで幅を作れる状態を目指す.

**データ分析での位置づけ**  
推論の柱その1. 効果の大きさを範囲で示し, 意思決定の安全マージンを確保する.

**6ステップの適用方針**  
- 「素朴な方法と限界」は**省略**する. 4章でSEの動機づけが十分に済んでいるため.  
- 問いの出発点は「前章で手に入れたSEと幅を, どう読み, どう報告するか」.

**章の構成**  
1. **問い**: 前章でSEを手に入れた. 「およそ2」の意味は何か. その幅をどう読み, どう報告するか. 平均以外にも使えるか.  
2. **点推定の限界**(短く): 1点の数字は「ばらつきの中の1点」. だから幅をつける.  
3. **信頼区間の定義と読み方**: 95%の正しい意味, よくある誤解, なぜ95%か.  
4. **3ケースの手計算**: 平均・割合・差. ここが章の実質的な本体.  
5. **幅を狭める戦略**: 件数・ばらつき・信頼水準. 件数別比較(4章から移動)をここで使う.  
6. **実務注意**: 偏りはnで解決しない(ギャラップ vs Literary Digest), 外れ値の影響, 対数変換(4章から移動).  
7. **結果と次の一手**: 幅をつけたが, 「差が偶然で説明できるか」にはまだ答えられない → 6章へ.

**コラム**  
推定量の性質(不偏性・一致性)を正確に扱うコラムを置く. 標本平均が母平均の不偏推定量であること, 不偏分散が n-1 で割られる理由を, 5章の流れの中で示す.

**4章から5章へ移動するトピック**  
- 件数別比較(n=5 vs n=100で幅が4倍違う話)→ 幅を狭める戦略の導入に使用  
- 対数変換の実践 → 実務注意として配置  
- ギャラップ vs Literary Digestの逸話は5章に残置(偏りはnで解決しない文脈で活用)

**5章から削除するトピック(4章と重複のため)**  
- SEの定義ボックスと部品の分解説明  
- SD vs SEの区別  
- √nで割る理由の解説  
- SEの前提条件(4章のCLT条件を参照する一文に置き換え)  
- 身長の例(10人 vs 100人のばらつき)  
- 12班の通学時間の例  
- 「素朴な方法と限界」セクション全体

**扱う根本概念**  
点推定, 信頼区間の意味と読み方, 平均の区間推定, 比率の区間推定, 差の区間推定.

**補足概念**  
予測区間, 標本サイズ設計, ブートストラップCI.

## 6. 検定 — 仮説と p 値をどう読むか(**二周構成**)
**物語の役割**  
「偶然でも起こりうる」を数で吟味し, 過剰な確信を避ける. まず道筋を一息で掴み, 次に代表的検定の数学的裏づけと計算を**手で**追える状態にする.

**データ分析での位置づけ**  
推論の柱その2. 事前に合意した閾値と手順で結論を出し, 説明責任を確保する.

### Part A: エッセンス(核だけ先に一周)
- 6ステップを一気通貫:  
  問い → 素朴な比較の限界 → 帰無 \(H_0\) と対立 \(H_1\), 検定統計量, p値, 有意水準 → 型(標本→統計量→\(H_0\) 下の珍しさ→結論) → 軽い実践 → 次の一手.
- 目的: 「検定は何をしているのか」を図1枚で説明できる状態へ.
- 個別検定の数学的根拠と数値計算は提示せず, Part Bに送る.

### Part B: Part Aの数学的補遺

**位置づけ**：Part Aで概念として紹介した道具について, 「なぜそうなるのか」を数学的に裏づける補遺. 6ステップ構成は適用しない.

**含める項目**：
1. t統計量の構造(なぜ標準誤差で割るか)
2. t分布の必要性(なぜ正規分布でなくt分布か)
3. 自由度の意味
4. Welch法によるt検定の数値計算(語学アプリ例の続き)
5. カイ二乗統計量の構造(なぜ (O−E)²/E か)
6. カイ二乗分布と自由度
7. カイ二乗検定の数値計算(血液型例の続き)
8. 第I種・第II種の誤りと検出力(Part Aの概念紹介を定式化)
9. 効果量(Cohen's d)

**等分散仮定について**: 本書ではWelch法を採用する. 母集団の等分散性を仮定することは, 母集団の情報を知ろうとする検定の目的と整合しないため. プールtには触れない.

**外に出すもの**：
- 推定量の不偏性・一致性 → 5章コラム
- カテゴリ変数の体系的扱い → 全体構成が固まった後で別途検討

**扱う根本概念**  
帰無・対立仮説, 検定統計量, p値, 有意水準, 第I種・第II種の誤り, 検出力, 効果量.

**補足概念**  
多重比較の注意, 正規性や等分散の形式検定, 置換検定の発想.

## 9. 統計の限界・バイアス・倫理 — 手法の外側を知る
**物語の役割**  
1章で「地雷がある」と予告し, 2〜8章で手法を積み上げてきた. その過程で各章は手法に固有の限界もその場で示している. 最終章はそれらを横断的に再整理し, 「地雷の地図」を渡す章. バイアスの類型を手法の前提と対応させながら整理し, 各章で散らばっていた限界を統合する. 注意事項の羅列ではなく, 「ここまで学んだことで何が見えるようになったか」を読者に実感させる締めくくりとして設計する.

**データ分析での位置づけ**  
品質保証とガバナンスの入口. バイアスの類型を手法の前提と対応させて整理し, 分析前(設計)・分析中(診断)・分析後(報告)で守る最低限のルールを示す.

**扱う根本概念**  
- **バイアスの類型と手法対応**: サンプリングバイアス(標本の偏り), 生存者バイアス(観測できないものの見落とし), 確証バイアス(解釈の歪み). 各バイアスが2〜8章のどの前提を崩すかを明示する.  
- **前提の確認**: 独立性, おおよその線形性, おおよその等分散性.  
- **誤用の代表型**: 過度な解釈, p値の誤読, 相関と因果の混同.  
- **データ倫理の基本**: 収集・利用・報告における最低限の規範.

**補足概念**  
多重比較の制御の枠組み, VIF と多重共線性, 事前登録と再現可能性, データガバナンスとプライバシ.

## 学習動線(章間の橋渡しの指針)
- **2→3**: 数で要約しても「形」は戻ってこない. 形を見ることで統計量で失われた構造に気づき, 分布として捉え直す.  
- **3→4**: 形の違いがなぜ推論に響くかを, 「取り直したらばらつく」の視点で一般化.  
- **4→5**: SEと「±約2SE」で幅の発想を手に入れた. 次はその幅をどう読み, どう使うか.  
- **5→6**: 幅だけでは決めきれない場面で, 偶然では説明しにくさを数で確かめる.  
- **6→7→8**: 関係の強さを測り, モデルとして表現し, 残差で安全確認.
