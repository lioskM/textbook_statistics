# 概念導入順序整理

## この文書の役割
本書のドラフト（chapters/ 配下の .tex）から, 各章で初めて導入される概念を本文の登場順に書き出す.
設計を先に決めるための文書ではなく, 完成済みドラフトの現状を追認し, 章横断の整合を確認するための表とする.

## 使い方
- 各章の欄に, その章で初出となる概念を, 本文に現れる順で並べる.
- 各概念に初出位置（節見出しなど）を添える.
- 確認の観点:
  - 未導入の用語が, 説明のないまま先の章・節で使われていないか.
  - 同一概念の呼称が全章で揃っているか.
- 記入は別タスクで行う. 本ファイルは現時点では枠のみとする.
