# 統計学教科書 修正方針

## 1. この文書の目的

- 作成日：2026-07-18
- 対象：`chapters/main.pdf` とそのLaTeX原稿
- 目的：次のセッションで、準拠チェックではなく内容の正確さと読了体験を基準に修正を再開できるようにする。

## 4. 修正の実施順序

局所的で影響範囲が明確な修正から行う。

1. 等分散性の説明を訂正する。
   - 対象は行番号ではなく、本文文字列「場所ごとに本当の散らばりが違うのに一律のSEで予測区間を出すと, 散らばりが小さい場所では狭すぎ, 大きい場所では広すぎる」で特定する。

   - 正しくは、一律の分散を使うと「散らばりが小さい場所では広すぎ、大きい場所では狭すぎる」。
   - 修正はこの反転した一文だけに限定する。「直線そのものは間違っていない」「平均の推定は使えても、予測の幅の保証は当てにならない」という周辺の書き分けは残す。

   - 「一様でないことが統計的に検出できる」ことと、「実戦で大きく勝率が上がる」ことを分ける。

3. 第1章に存在しない仮定を解消する。

   - 両箇所は「第1章で置いた、標本は大きな偏りなく抽出されたという仮定」を参照するが、第1章にその文はない。

   - 有効性評価では統計的有意性が重要な判断材料になるが、承認は試験設計、安全性、効果の大きさ、再現性などを含む総合判断だとする。

5. 第4章実践の再抽出手続きを明示する。

   - 80本を経験分布とみなし、毎回復元抽出で20本を選ぶのか、非復元の部分標本を選ぶのかを明記する。

### 第2群：分散の定義と数値の一本化

1. 第2章では、手元のデータを記述する分散として `n` で割る定義を残す。
2. 推論へ入る第4章で、母分散推定に使う分散は `n-1` で割ることを明示する。
3. 第2章の値が「誤りだった」と見えないよう、目的が違う二つの定義だと説明する。
4. 記号 `s^2` を両方に使うと定義が衝突するため、同じ記号を文脈だけで使い分けない。対象には第4〜7章だけでなく、第8章8.3で既に `n-1` 割りとして定義している標本共分散 `s_{xy}` と標本標準偏差 `s_x`, `s_y` も含める。第2章、第4〜7章、第8章8.3を、以下で確定した記号方針に従って同じ記号監査・数値再計算パスで扱う。
5. 第7章では、自由度の導出を通じて `n-1` の理由を回収する。第4章で結果を先に使い、第7章で理由が分かる構造にする。
6. 不偏分散側の記号は、統計WEBの[標本分散と不偏分散](https://bellcurve.jp/statistics/course/8614.html)と[共分散は「n」で割る？「n-1」で割る？](https://bellcurve.jp/statistics/blog/13480.html)における使い分けを照合したうえで確定した。外部サイトの記号をそのまま採用せず、本書で第2章から第8章まで読み分けられることを判断基準にする。

記号方針は次のとおり確定した。

- 第2章で手元のデータを記述する `n` 割りの分散は `s^2`、標準偏差は `s` を維持する。
- 第4章以降で母分散の推定に使う `n-1` 割りの不偏分散は `u^2`、その平方根は `u` とする。
- 第8章8.3の `n-1` 割りの標本共分散と標本標準偏差は、`u_{xy}`、`u_x`、`u_y` とする。
- 相関係数は `r = u_{xy}/(u_x u_y)` と書く。分子と分母に共通する `n-1` が約分されるため、記号を変更しても相関係数の値は変わらないことを本文に一言添える。

判断記録：統計WEBが不偏分散に `s^2` を当てる流儀であっても、本書では既習記号の意味を途中で変えることによる読者の混乱を避ける判断を優先し、不偏分散に `u^2` を採用する。

判断記録：本文では「経験分布」の用語導入を廃止し, 説明文（80本のどれも同じ確率で選ばれる仮の分布）で通す. 理由: 定義は明確だが実務でほぼ流通しない用語であり, 想定読者に覚えさせる便益がないため.

判断記録：じゃんけんの出典はページ番号なしの書籍出典＋doi付き論文併記で完成とし, ページ照合は実施しない. 理由: 書籍・論文とも現物照合の手段がなく, 現行の形で出典として成立しているため.

CSVから確認済みの値は次のとおり。

- 中央値：107.084万回
- 推論用標準偏差（`n-1`）：240.785万回
- 製作費と視聴回数の相関：0.3650
- おすすめ表示回数と視聴回数の相関：0.8033
- 平均評価と視聴回数の相関：0.2057
- 原作あり：37本、平均274.082万回、標準偏差260.637万回
- 原作なし：43本、平均153.717万回、標準偏差209.680万回
- 平均差：120.365万回
- 差のSE：53.464万回
- Welchのt値：2.251
- Welch–Satterthwaite自由度：68.936

上記を基準に、次を一回の再計算パスで直す。

- 第4章のSE
- 第5章の平均・群差の信頼区間
- 第6・7章のt検定と効果量

再計算はスクリプト化し、丸め前の値を一か所から取得できるようにする。手計算で章ごとに別の値を入力しない。

実行順は次のとおりとする。

1. まず `data_generation/` に再計算スクリプトを作り、丸め前の値を一元管理する。本文はまだ変更しない。
2. スクリプトの出力が、少なくとも `u = 240.785`、群別の `u = 260.637 / 209.680`、差のSE `53.464`、Welchのt値 `2.251`、Welch--Satterthwaite自由度 `ν = 68.936`、相関係数 `0.3650 / 0.8033 / 0.2057` と一致することを確認する。
3. 基準値との一致を確認してから、記号置換と数値置換を同一の監査・再計算パスで実施する。記号だけ、または数値だけを先に一括置換しない。
4. 第9章の仮置きデータをCSV実値へ差し替える際は、第10章10.4で再掲している回帰式、製作費の範囲、切片・傾き・予測値を同じ出力から連動して修正する。80本の実値では、製作費を億円、視聴回数を万回として `y \approx 108.4 + 49.4x`、観測範囲は `0.25〜11.46` 億円、`R^2 = 0.3650^2 \approx 0.13` となる。
5. 第1群からの波及確認として、第10章10.1の標本に関する記述と、第4章実践の再抽出手続きを再確認する。2026-07-18時点で本文側は修正済みである。第4章の生成スクリプトと仮図は非復元抽出のままなので、第3群で復元抽出へ変更して再生成する。

第9章の実践は、10行の仮置き表と残差表を廃止し、80本のデータから再計算した偏差積和と偏差平方和を公式へ代入する形にする。残差は表にせず、第3群で作る残差プロットで形を確認する。本文では、製作費だけでは視聴回数の散らばりの1割強しか説明できないことを軸に、弱い当てはまりを「引いた線を疑う」という章の主題へつなぐ。外挿は観測範囲外の30億円を例にする。

### 第4群：内容と読了体験の通読修正
