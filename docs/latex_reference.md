# LaTeX リファレンス

## ドキュメント構成

- ドキュメントクラス: `jsarticle`（platex）
- 章の単位: `\section`（jsarticleのため `\chapter` は不可）
- 各章ファイルは `\section` から始まる断片。`main.tex` から `\input` する
- スタイル定義: `pre.sty`（tcolorboxベースのカスタム環境）

## カスタム環境一覧（pre.sty定義）

すべてのボックス環境は `defi` のカウンタを共有し、通し番号が振られる。

| 環境名 | 用途 | 書式 | ラベル接頭辞 |
|--------|------|------|-------------|
| `defi` | 定義 | `\begin{defi}[opt]{用語名}{ラベル}` | `thm:ラベル` |
| `thm` | 定理 | `\begin{thm}[opt]{定理名}{ラベル}` | `thm:ラベル` |
| `prop` | 命題 | `\begin{prop}[opt]{命題名}{ラベル}` | `thm:ラベル` |
| `lem` | 補題 | `\begin{lem}[opt]{補題名}{ラベル}` | `thm:ラベル` |
| `coro` | 系 | `\begin{coro}[opt]{系名}{ラベル}` | `thm:ラベル` |
| `rmk` | 注意 | `\begin{rmk}[opt]{タイトル}{ラベル}` | `thm:ラベル` |
| `ex` | 例 | `\begin{ex}[opt]{タイトル}{ラベル}` | `thm:ラベル` |
| `axiom` | 公理 | `\begin{axiom}[opt]{公理名}{ラベル}` | `thm:ラベル` |
| `pform` | 演習問題 | `\begin{pform}[opt]{タイトル}{ラベル}` | `thm:ラベル` |
| `mybox` | コラム等 | `\begin{mybox}[opt]{タイトル}` | なし |
| `tcolorbox` | 汎用ボックス | `\begin{tcolorbox}[オプション]` | なし |

### 注意
- `\begin{dfn}` は使用しない（旧形式）。すべて `\begin{defi}[opt]{用語}{ラベル}` を使う
- 用語名が空の場合は `{}` を渡す（内部で `\kara` と比較して表示を制御）
- 定義ブロック内には前提・記号・式・名称のみを書く（リズム原則R9）
- 参照は `\cref{thm:ラベル}` で行う
