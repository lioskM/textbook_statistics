# ランニング例設定メモ(改訂版)

統計学入門教科書(人文系大学生・社会人向け)で全章を通して使用するランニング例の設定.

---

## 1. 例の設定

### 舞台
あるサブスク型動画配信サービス. 実在のサービスは特定しない.

### 対象データ
そのサービスから, 分析用に抽出された **N本(暫定: 80本程度)** の邦画(実写)作品の標本データ. 公開年は揃えてあるが, サービス全体には数百本の対象作品があり, 今回の標本はその一部である.

この設定により, 5章以降の推測統計(区間推定・検定)の動機が自然に立つ. 「手元のデータは全体の一部であり, ここから観測されていない範囲について何が言えるかを考える」という構造を最初から明示する.

「邦画(実写)・特定の年」という範囲そのものは設計上の制約として前面に出さない. 9章・10章で「この範囲の外について何も言えていない」という気付きを読者側に起こさせる.

### 変数

**量的変数**
- 公開後30日間の視聴回数(主たる被分析変数)
- 製作費(円)
- 平均評価(5段階の平均値)
- 評価件数(レビュー数)
- おすすめ表示回数(サービス内でユーザーに表示された回数)
- 上映時間(分)

**質的変数**
- ジャンル(サスペンス, コメディ, 恋愛 など)
- 原作の有無(オリジナル脚本 / 既存作品の映画化)

データはすべて架空. 値域・分布形は教育的に扱いやすい範囲で設計する(具体値は別途).

### 視聴回数の定義について

主変数として用いる視聴回数は **「公開後30日間の累計」** に揃える. 単純な「該当年内累計」を採用しない理由は, 1月公開と12月公開で観測期間が異なり, 比較が歪むためである.

なお「年内累計視聴回数」は, 10章で **「集計期間を揃えないと比較が歪む」** という悪い設計の例として明示的に扱う. これにより測定設計の重要性を体感させる.

### 標本抽出に関する仮定

5章以降の推測統計を成立させるため, 本書では次の仮定を最初に置く.

> ここではまず, この標本が対象作品全体から大きな偏りなく抽出されたものとして扱う. ただし, その仮定がどこまで成り立つか, またこの「対象作品全体」をどこまで外の世界へ広げてよいかは, 9章・10章で改めて検討する.

この一文により, 推測統計の動機が成立し, かつ9-10章での批判的検討への伏線となる.

## 2. 各章での使われ方

| 章 | 内容 | この例での扱い |
|---|---|---|
| 2章 データの要約 | 代表値・散らばり | 視聴回数の平均・中央値・分散・標準偏差. 平均と中央値が大きく異なる(右に歪む)ことから, 代表値の選択が見え方を変えることを体感する. |
| 3章 データの可視化 | ヒストグラム・散布図・箱ひげ図 | 視聴回数のヒストグラム(右に歪む). 平均評価のヒストグラム(J字型). 製作費 vs 視聴回数の散布図. ジャンル別の視聴回数の箱ひげ図. **評価件数 vs 平均評価の散布図(評価件数が少ない作品の平均評価が不安定であることを視覚化)**. |
| 4章 確率分布と中心極限定理 | 正規分布・CLT・標準誤差 | 「全作品リストから20作品を無作為に選んで平均視聴回数を計算する」という調査を繰り返したらどうなるかを考える. 標本平均の分布が正規分布に近づくことを示す. サブ例としてSNS視聴時間を補助的に用いる. |
| 5章 推測統計の入口 | 母平均の区間推定 | 手元の標本から, 同条件の作品群全体の平均視聴回数を区間推定する. 「全部わかっているのに推定する」のではなく, 「一部しか手元にないから全体を推測する」という構造を最初から明示する. |
| 6章 仮説検定 | 2群の平均の差 | **「原作あり作品と原作なし作品で平均視聴回数に差があるか」を検定する**. 群の境界が明確で初学者に扱いやすい. ジャンル比較は3章の箱ひげ図(複数群の散らばりの可視化)に回す. |
| 7章 相関 | 相関係数・散布図 | 製作費と視聴回数の相関. 加えて, **おすすめ表示回数と視聴回数の相関, 平均評価と視聴回数の相関を比較する**. 「視聴回数は作品の魅力だけでなく, 配信側の配置にも左右されるかもしれない」という気付きを誘導する. **おすすめ表示回数と視聴回数の関係は「表示されたから見られた」とも「見られたからさらに表示された」とも読めることに触れ, 相関だけでは関係の向きが決められないことを明示する**. |
| 8章 回帰分析 | 単回帰 | 製作費から視聴回数を予測する単回帰モデル. 予測誤差が大きいことを示し, 「本当はおすすめ表示回数や評価件数も入れたい」という不足感を読者に持たせる. これがより進んだ回帰分析への動機になる. |
| 9-10章 統計の限界・バイアス・倫理 | 測定・サンプリング・解釈の限界 | 「何秒で1視聴か」の測定問題. 推薦アルゴリズムによる露出バイアス(視聴回数は作品の質ではなく, 表示機会も反映している). 「邦画・実写・特定の年」というサンプリングの限界. 「年内累計視聴回数」を使うと公開月で比較が歪む(集計期間設計の重要性). |

## 3. 各章末の小コメント

9-10章で批判的視点が突然出てきて前章までを否定する形になることを避けるため, **各章末に「ここで言えたこと / まだ言えないこと」を短く置く**.

例(2章末):
> 平均視聴回数は, このデータ内で作品がどの程度見られたかを要約している. ただし, それが作品の質や満足度を直接表しているわけではない.

このような注意書きを早い段階から積み上げることで, 9-10章の批判的議論が「前章の否定」ではなく「前章で気になっていた点の正面からの検討」として接続する.

---

## 4. ゆるい目的設定

強い目的変数を立てず, 「視聴回数の増加につながるヒントが見つかったかもしれない」という慎重なトーンで進める.

- 7章までは目的を定めず, 変数間の関連や分布の探索として進める.
- 8章で「視聴回数を製作費から予測したい」という方向性を導入する.
- 9-10章で「予測ができたとして, それは増加施策の根拠になるか」を批判的に検討する.

このトーンは, 「統計の結果はしばしば素朴な期待への部分解にとどまる」という本書の柱(原則4)と整合する.

## 5. 改訂履歴(初版からの主な変更点)

1. **データ設定**: 「全数データ」と読まれる書き方を改め, 「分析用に抽出された標本」であることを最初から明示. 推測統計の章で動機が自然に立つようにした.
2. **視聴回数の定義**: 「年内累計」から「公開後30日間累計」に変更. 公開月による比較の歪みを回避. 「年内累計」は10章で悪い設計例として活用.
3. **変数追加**: おすすめ表示回数, 評価件数を追加. 7章の相関比較・3章の可視化・9-10章の批判的議論を強化.
4. **6章検定主例**: 「コメディ vs サスペンス」から「原作あり vs 原作なし」に変更. 群の境界を明確にした.
5. **各章末コメント**: 「言えたこと / まだ言えないこと」を各章末に置く方針を追加.
6. **標本抽出の仮定**: 推測統計を成立させるため, 「標本は対象作品全体から大きな偏りなく抽出されたものとして扱う」仮定を1節末尾に明記.
7. **7章の因果方向の注意**: おすすめ表示回数と視聴回数の関係について, 相関の向きが決められないことを明示する方針を追加.
