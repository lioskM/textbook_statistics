\section{推測統計 --- 手元のデータから全体を考える}

\subsection{Question：このデータは全体の代表と言えるか？}

100人にアンケートをとって「みんなの意見」を語る, 数日分の売上データから「今後のトレンド」を予想する――  
こうした行為は, 私たちの日常にごく自然に紛れ込んでいる.  
手元の一部のデータ（標本）から, 大きな全体（母集団）の姿を推し量ろうとするのだ.

しかし冷静に考えれば, これはかなり大胆な試みだ.  
母集団の全員を調べたわけではなく, そのごく一部しか知らないのに, 「これで全体がだいたいわかる」と思いたくなってしまう.  
では, その「一部」から, どこまで全体を語ることができるのだろうか.  
この問いを起点に, 私たちは推測統計の世界に踏み込む.

\subsection{Naive Approach：素朴な方法と限界}

もっとも単純なやり方は, 集めた標本の平均や割合を, そのまま母集団の平均や割合とみなすことだ.  
計算も直感もシンプルで, 一度試せばすぐに理解できる.

しかし, この方法には致命的な弱点が潜む.
\begin{itemize}
\item 標本がたまたま偏っているかもしれない
\item 標本数が少ないと, 偶然の影響で値が大きくぶれる
\item 抽出の仕方によっては, 母集団を正しく反映しない
\end{itemize}

たとえば, 道端で会った10人の身長平均を「この街の平均身長」と言い切ってしまうようなものだ.  
簡単ではあるが, これでは「偶然のいたずら」を見抜く術がない.  
この限界が, 次に進むための理論的な枠組みを必要とさせる.

\subsection{Theory Kernel：本質的な課題と解決の方向}

全員を調べることはほぼ不可能だが, それでも全体像を知りたい――この動機が推測統計を動かす.  
そこで私たちは部分（標本）を頼りに全体（母集団）を描こうとするが, その過程には必ず「偶然のゆらぎ」がつきまとう.

同じ母集団から同じ方法で標本を何度も取り直す場面を想像してみる.  
そのたびに計算される平均や割合は少しずつ異なるだろう.  
こうして得られる値をすべて並べたときに現れる分布――これが標本分布であり, ゆらぎの正体である.  
私たちの手元にある推定値は, あり得た多くの値の中のひとつにすぎない.

では, このゆらぎの大きさをどう測るか.  
その答えが標準誤差（Standard Error, SE）である.  
SEは標本の標準偏差を標本数の平方根で割った形を持ち, これは「データが多いほど偶然の影響は相殺され, 推定が安定する」という直感と, 平均の分散が標本数に反比例するという確率論の法則を反映している.

もちろん, これが成り立つには条件がある.  
標本は無作為に抽出され, 各観測が独立し, 分布の性質が安定している必要がある.  
この条件が崩れれば, 計算結果は数字として存在しても信頼には足らない.

信頼区間（confidence interval, CI）は, このSEと標本分布の形を使って構築される.  
推定値の周囲に「真の値を捕まえられる範囲」を設け, その幅が不確実性の大きさを示す.  
これは1930年代にネイマンが提唱した枠組みであり, 当時フィッシャーとの間で「確率の意味」を巡る激しい論争を呼んだ.  
偶然をどう数値化するかというこの格闘が, 現代の推測統計の姿を形作ったのである.

例えば標本平均170cm, SEが2.5cmの場合, 母集団の標準偏差が既知なら正規分布に基づき
\[
170 \pm 1.96 \times 2.5 \ \Rightarrow\ 165.1\ \text{cm} \ \text{〜} \ 174.9\ \text{cm}
\]
と計算できる.  
だが現実には母集団のばらつきは未知であり, その推定値自体にも不確実性があるため, より慎重なt分布を使う必要がある.  
この違いが, 次の具体例で使う数字の差に現れる.

\subsection{Method：区間推定の要点}

区間推定の核は, 「推定値のまわりに不確実性の幅を設ける」ことに尽きる.  
幅を決める要因は, 標本のばらつきと標本数の2つであり, この点はTheory Kernelですでに説明した.  
したがって, ここでは詳細な式の再掲はせず, 具体的な計算例でその意味を確かめることにする.

\subsection{Practice：例で確かめる}

ある商品の重量を64個測定したところ, 平均170g, 標準偏差12gだった.  
標準誤差は $12/\sqrt{64} = 1.5$ g となる.  
ここで母集団の標準偏差は未知であるため, 正規分布の1.96ではなく, 自由度63のt分布から得られる約2.00を使う.  
これは, ばらつきの推定値に含まれる不確実性を反映させるためだ.

計算すると, 信頼区間は
\[
170 \pm 2.00 \times 1.5 \Rightarrow 167.0\ \text{g} \ \text{〜} \ 173.0\ \text{g}
\]
となる.  
この幅を示すことで, 推定の確からしさを具体的に共有できる.

\subsection{Output \& Next：結果と次の一手}

ここまでで, 
\begin{itemize}
\item 標本から得られる値には必ず偶然のゆらぎがある
\item ゆらぎは標本分布とSEで数値化できる
\item CIとして幅を示せば推定の信頼度を明確に共有できる
\item 実務では母集団の標準偏差は未知であり, その場合はt分布を用いる
\end{itemize}
ということがわかった.

では, このゆらぎはどのような形をしているのか.  
もし標本平均の分布が特定の形に収束するのなら, その形は推測や検定の土台となるはずだ.  
次章では, この「ゆらぎの形」＝分布の構造を探り, なぜ標本平均が正規分布に近づくのか, そしてそれが統計的推論にどのような力を与えるのかを明らかにする.
