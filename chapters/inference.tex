\section{手元のデータから全体を考える}

  \subsection{問い : 図で見えた“形”はこの先も言えるのか}
    前章で図にしてみると, いくつかの手がかりが見えたはずだ.  
    レビュー数のヒストグラムは右に裾が長いように見える.  
    カテゴリー別の箱ひげ図では, あるカテゴリーの購買数が高そうに見える.  
    散布図では, レビュー数が増えるほど購買も増えるように見える.

    では, これは\textbf{たまたま}なのか, それとも\textbf{この集団全体でも言える傾向}なのか.  
    もし同じ集団からもう一度データを集めたら, どれくらい同じ形になるだろうか.  
    今見えた数字に, どれくらいの\textbf{幅}をつけて語れば言い過ぎにならないだろうか.

    私たちはしばしば, 手元の一部のデータ（標本）から, 見えない全体（母集団）を推し量る.  
    図で得た手がかりを, \textbf{幅をつけて}全体の話に持ち出すにはどうすればよいか.  
    ここからが, 推測統計の出番である.

  \subsection{素朴な方法 : 1回の数字に惚れない}
    いちばん簡単なのは, 標本の平均や割合を, そのまま全体の数字だと見なすこと.  
    けれど, これは\textbf{たまたま}に弱い. 標本が偏っていたり, 数が少なかったり, 取り方に癖があったりすると, 1回の数字は大きくぶれる.

    \textit{なぜダメか（可視化と結びつけて）}:
    \begin{itemize}
      \item \textbf{少ない観測は揺れが大きい}. ヒストグラムはビンの切り方ひとつで山が動き, 散布図は1点の外れ値に引っ張られる. 少数の平均は\textbf{偶然の波}を拾いやすい.
      \item \textbf{取り方の偏り}. 朝だけの来店データで「1日の平均」を語る, レビューを書きやすい人だけが残る, といった\textbf{集め方の癖}で数字が寄る.
      \item \textbf{時間が混ざる}. たまたまキャンペーン期間だった, 季節要因が乗っていた. 今見えた形が\textbf{次回も同じ}とは限らない.
      \item \textbf{互いに似たデータ}. 友人紹介で似た顧客が続けて現れる, 同じ人が何度も登場する. 観測同士が\textbf{独立でない}と, 実質の情報量は少ない.
      \item \textbf{測り方の揺れと外れ値}. 計測ミスや異常値が平均を引っ張る. 可視化では点やヒゲで気づけるが, \textbf{平均に即落とす}と見誤る.
    \end{itemize}

    だから必要なのは, \textbf{ぶれを見積もって, ぶれ込みで話す}という発想だ.

  \subsection{核となる考え : 取り直せば毎回ちがう → ゆらぎを数にする}
    同じ集団から, 同じやり方で標本を何度も取り直すところを想像してみよう.  
    そのたびに計算する平均や割合は, 少しずつ違うはずだ. これが\textbf{ゆらぎ}である（この並びの形を, 後で「標本分布」と呼ぶ）.  
    私たちの手にある推定値は, その並びの中の\textbf{たった1点}. だから\textbf{幅をつけて}語る必要がある.

    幅の大きさを見積もる物差しが\textbf{標準誤差}（SE）だ.  
    SEは「推定値がどれくらい揺れそうか」の\textbf{目安}. 平均の場合は
    \[
      SE(\bar{x}) \approx \frac{s}{\sqrt{n}}
    \]
    で考える（$s$: 標本の散らばりの大きさ, $n$: 標本数）.  
    観測が増えると $1/\sqrt{n}$ のペースで\textbf{安定}していく —— 直感にも合う.

    \textit{前提のメモ}: 無作為に選ぶ, 互いに影響し合わない（ほぼ独立）, 同じ集団から来ている.  
    ここが崩れると, 数字は出せても\textbf{当てにならない}.

    \textit{SDとSEのちがい}:  
    SD（標準偏差）は\textbf{データそのものの散らばり}, SEは\textbf{推定値の散らばり}.  
    データが多様でも, $n$ を増やせば推定は安定しうる（SEは小さくなる）.

  \subsection{やり方 : 数字に「幅」をつけて言う}
    幅のつけ方はたったこれだけ:
    \[
      \text{推定値} \ \pm \ (\text{およそ}2) \times SE \qquad \text{（95\%の目安）}
    \]
    これが\textbf{信頼区間}の考え方の芯である. 小さな標本では, 「2」を\textbf{少しだけ大きめ}（たとえば 2.1 〜 2.3）にすることがある.  
    道具（表計算や統計ソフト）はこの調整を自動でやってくれる. ここでは\textbf{考え方}をつかむことを優先する.

    \textit{どう読むか（誤解しやすいポイント）}:  
    「95\%」は\textbf{方法の当たりやすさ}を表す. 同じ手順で何度もやれば, だいたい95\%の幅が真の値を含むという約束だ.  
    「この幅の中に真の値が95\%の確率で入る」とは言わない（真の値は固定で, 当たるか外れるかの2択だから）.

  \subsection{体験 : 手を動かして3ケース}
    \paragraph{1) 平均に幅をつける}
    ある商品の重量を $n=64$ 個測ると, 平均 $170$ g, 標準偏差 $12$ g.  
    SEは $12/\sqrt{64}=1.5$ g. 95\%の目安は
    \[
      170 \pm 2 \times 1.5 \ \Rightarrow\ 167.0\ \text{g} \ \text{〜} \ 173.0\ \text{g}.
    \]
    \textit{読み方}: 「このやり方を繰り返せば, こういう幅がだいたい当たる. 今回の幅はその1本.」

    \paragraph{2) 割合に幅をつける}
    20商品のうち, 星評価が 4.0 以上のものが $k$ 品だったとする（$k$ は後で挿入）.  
    割合のSEはおおよそ
    \[
      SE(p) \approx \sqrt{\frac{p(1-p)}{n}} \quad (\;p=k/n\;)
    \]
    で見積もる. 端に寄る（0や1に近い）ときは, この式が\textbf{楽観的になりがち}なので, 道具は\textbf{少し広めの安全な幅}を自動で作る.  
    考え方は同じで, \textbf{推定値} $\pm$ \textbf{およそ2} $\times SE$ で読む.

    \paragraph{3) 2つの平均の差に幅をつける}
    カテゴリAとBで購買数の平均を比べたい（各群の平均 $\bar{x}_A, \bar{x}_B$, 散らばり $s_A, s_B$, 件数 $n_A, n_B$）.  
    差のSEは
    \[
      SE(\bar{x}_A-\bar{x}_B) \approx \sqrt{\frac{s_A^2}{n_A}+\frac{s_B^2}{n_B}}.
    \]
    あとは\ \ \textbf{差} $\pm$ \textbf{およそ2} $\times SE$.  
    \textit{読み方}: 件数が多い群や散らばりが小さい群は, 自然に\textbf{重みが大きく}効く（式がそうなっている）.

    \vspace{0.5em}
    \textit{幅を狭めたいときの3つのつまみ}:
    \begin{itemize}
      \item \textbf{数を増やす}（$n$ を増やすと $SE$ は $1/\sqrt{n}$ で縮む）.
      \item \textbf{測り方を安定させる}（ばらつき $s$ を下げる, 層化するなど）.
      \item \textbf{自信の度合いを下げる}（95\% → 90\% にすれば幅は狭くなるが, 外しやすくもなる）.
    \end{itemize}

  \subsection{作法 : よくある誤解と現場の注意}
    \begin{itemize}
      \item \textbf{幅の誤読}: 「この幅の中に95\%の確率で入る」ではない. 95\%は\textbf{方法の約束事}.
      \item \textbf{SDとSEの混同}: SDは\textbf{データの散らばり}, SEは\textbf{推定の散らばり}. 役割が違う.
      \item \textbf{偏りの放置}: 無作為でないと, $n$ を増やしても\textbf{自信満々に間違える}. 取り方をまず点検する.
      \item \textbf{外れ値への弱さ}: 平均とSDは外れ値に弱い. きつい外れがあるときは中央値や分位で要約し, 幅づけは\textbf{ブートストラップ}も検討（道具がやってくれる）.
      \item \textbf{手順の透明性}: 期間や条件の切り取り, 欠損の扱いは幅に効く. ノートに\textbf{明示}しておく.
    \end{itemize}

  \subsection{結果と次の一手 : 幅で語れたら, 次は「偶然か」を問う}
    ここまでの核:
    \begin{itemize}
      \item 取り直せば数字は揺れる → \textbf{幅をつけて話す}.
      \item 幅の物差しは\textbf{標準誤差}. 平均なら $s/\sqrt{n}$ が目安.
      \item 幅の作り方は\textbf{推定値 ± およそ2×SE}（95\%の目安）.
    \end{itemize}
    次は, 見えた「差」が\textbf{偶然のぶれで説明しにくい}と言えるのかを問う.  
    その前に, なぜ「およそ2」でうまくいくのか —— 多く集めると\textbf{形が安定してくる}という事実を, 次章で確かめる.

  \subsection*{作業 : ランニングデータで幅をつけて言う（値は後で挿入）}
    ECサイトの20商品データについて, 次を計算し, 図（誤差棒）と一文の所見を残せ（小数第2位, 単位併記）.
    \begin{enumerate}
      \item \textbf{購買数の平均の幅}（95\%目安）: $\bar{x}, s, n, SE$, 幅＝$\bar{x}\pm 2\times SE$.
      \item \textbf{星評価 \,$\ge 4.0$\, の割合の幅}（95\%目安）: $k, n, p=k/n$, $SE\approx \sqrt{p(1-p)/n}$, 幅＝$p\pm 2\times SE$（端に近いときは道具の安全側の幅で上書き）.
      \item \textbf{カテゴリーAとBの平均差の幅}（95\%目安）: 差, $s_A, s_B, n_A, n_B$, $SE\approx \sqrt{s_A^2/n_A+s_B^2/n_B}$, 幅＝差$\pm 2\times SE$.
    \end{enumerate}
    \textit{記録の約束}: 抽出条件（対象と期間）, 欠損や外れ値の扱い, 切り取り基準をノートに明示する.
